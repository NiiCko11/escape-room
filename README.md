# 🖥️ El Servidor Perdido

Proyecto web tipo Escape Room desarrollado con PHP, MySQL, JavaScript y CSS.

El objetivo del juego es recuperar el acceso a un antiguo servidor resolviendo pistas digitales. Cada respuesta correcta desbloquea el siguiente nivel hasta completar la misión.

---

# 📖 Historia

📡 TRANSMISIÓN ENTRANTE — AÑO 2035

Un antiguo servidor de respaldo ha sido encontrado en una sala abandonada del datacenter.

Para recuperar los archivos debes superar una serie de desafíos digitales.

Cada respuesta correcta desbloqueará el siguiente nivel.

¿Podrás recuperar el acceso antes de que el servidor se apague?

---

## ⚙️ Instalación

1. Clonar el repositorio:

```
git clone https://github.com/NiiCko11/escape-room
```

2. Colocar el proyecto en:

```
C:\xampp\htdocs\
```

3. Importar la base de datos:

* Abrir phpMyAdmin
* Crear base de datos: `escape-room`
* Importar archivo:

```
/database/escape_room.sql
```

4. Ejecutar en el navegador:

```
http://localhost/escape-room/index.php
